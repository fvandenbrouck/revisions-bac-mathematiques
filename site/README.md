# Site de révisions du bac — spécialité mathématiques

Structure prévue :

- `site/index.html` : application statique.
- `site/pdf/` : sujets PDF.
- `site/programme/` : programme officiel.
- `site/data/` : données JSON générées.
- `site/img/` : pages de sujets rendues en images.

Test local :

```bash
cd site
python -m http.server 8000
```

Puis ouvrir `http://localhost:8000/`.
